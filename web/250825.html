확장프로그램 및 VS-Code에서의 html 작성

- open in browser : html 파일을 브라우저에서 alt B로 실행 가능하게 해주는 확장 프로그램

- auto rename tag : 태그 수정 시, 세트로 일치하는 태그도 한 번에 수정해주는 확장 프로그램

- highlight matching tag : 서로 일치하는 열림태그와 닫힘태그를 표시해주는 확장 프로그램

- live server : html 파일을 우클릭하여 실시간으로 반응하는 페이지를 켜주는 확장 프로그램

- '!'를 입력하여 vs-code에서 최소한의 설정을 자동으로 생성 가능

HTML

- HTML은 텍스트의 구조이자 의미를 나타냄

- Emphasis : 기울임 - em, 볼드체 - strong

- 하이퍼링크 : a - href 속성 필수
